EMBEDDED ENGINEER

  • Bachelor's Degree
  • Full-Time
  • Cyberjaya

Job Description

eMooVit is specialised in providing autonomous driving solutions. The company is in Cyberjaya with business focus on a complete solution for last mile autonomous mobility and robotics in agriculture. You will work with the Development team to design, coding, implementation & testing of new features or enhancements for embedded software applications.

Responsibilities

  • Participate in design, coding, implementation & testing of new features or enhancements for software in embedded applications.
  • Build reusable code & libraries for various projects.
  • Perform troubleshooting and identification of root causes to instabilities and errors.
  • Document and record the design process in a timely manner.
